# 宽频谐波页面获取C数据位置说明

## ETS 页面里和 C 交互的关键位置

- `entry/src/main/ets/pages/broadband_harmonic.ets:4`
  - `import * as harmonicTelemetryNativeModule from 'libentry.so';`
  - 这里是页面直接引入 C/NAPI 模块的入口。

- `entry/src/main/ets/pages/broadband_harmonic.ets:633`
  - `loadHarmonicDisplayNativeModule()`
  - 这里负责拿到 NAPI 模块实例，优先用 `libentry.so`，拿不到再走 `requireNapi(...)`。

- `entry/src/main/ets/pages/broadband_harmonic.ets:669`
  - `nativeModule.requestHarmonicDisplayData();`
  - 这里是向 C 层发起一次“请求刷新谐波数据”。

- `entry/src/main/ets/pages/broadband_harmonic.ets:699`
  - `voltageValue = nativeModule.getHarmonicVoltageDisplayData();`
  - 这里是从 C 层读取电压谐波数据。

- `entry/src/main/ets/pages/broadband_harmonic.ets:718`
  - `currentValue = nativeModule.getHarmonicCurrentDisplayData();`
  - 这里是从 C 层读取电流谐波数据。
