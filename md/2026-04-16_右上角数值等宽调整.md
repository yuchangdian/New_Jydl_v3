## 调整内容

本次把右上角主数据显示区的数值样式统一成了“固定单元格宽度 + 等宽字体”。

涉及区域：

- 电压
- 保护电流
- 测量电流

## 修改方式

在 `entry/src/main/ets/pages/Index.ets` 中新增了常量：

```ts
private readonly rightSectionValueFontFamily: string = 'monospace';
```

然后把上述几组数值文本统一补上：

```ts
.width(this.rightSectionValueWidth)
.fontFamily(this.rightSectionValueFontFamily)
.textAlign(TextAlign.Start)
```

## 结果

这样处理后：

1. 各个数值单元格继续使用同一套固定宽度
2. 数字和符号本身改为等宽字体显示
3. 数值变化时，视觉上会更整齐，不会因为某些字符本身宽窄不同而显得参差

## 备注

当前代码里的标题是“测量电流”，所以本次实际改动覆盖的是右上角“电压、保护电流、测量电流”这三组数值区。
