## 调整内容

本次只调整了软压板区域的竖向间距，没有改复选框逻辑，也没有改硬压板区域。

## 修改方式

在 `entry/src/main/ets/pages/Index.ets` 里新增了一个常量：

```ts
private readonly softPlateVerticalSpace: number = 8;
```

然后把软压板四列列表从：

```ts
Column()
```

改成了：

```ts
Column({ space: this.softPlateVerticalSpace })
```

## 结果

这样每一列软压板项之间都会多出统一的竖向间距，看起来会比原来松一些。

如果后面还想再大一点，直接继续调：

```ts
softPlateVerticalSpace
```

比如改成 `10` 或 `12` 即可。
