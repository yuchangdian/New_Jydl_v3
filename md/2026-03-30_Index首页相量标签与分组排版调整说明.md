# Index首页相量标签与分组排版调整说明

## 调整内容

- 首页“保护电流”“测量电流”标签改为加粗显示
- 这两个标签与“电压”使用同一套固定宽度标签列，左边界对齐
- 电压、保护电流、测量电流的相量值改为按组排版
- 每个 `U` / `I` 相量文本作为一个整体显示
- 相量组之间保留一段固定的小间距，不再用整句字符串里的空格硬拼

## 修改文件

- `entry/src/main/ets/pages/Index.ets`

## 实现方式

- 新增 `PhasorSectionLabel(label: string)`
- 新增 `PhasorGroup(prefix: string, value: string)`
- 新增首页相量排版常量：
  - `INDEX_PHASOR_LABEL_WIDTH`
  - `INDEX_PHASOR_LABEL_GAP`
  - `INDEX_PHASOR_GROUP_GAP`

## 结果

- “电压”“保护电流”“测量电流”三行标签样式统一
- `Uₐ ... / Uᵦ ... / U꜀ ...`、`Iₐ ... / Iᵦ ... / I꜀ ...` 会按组分开显示
- 后续如果还要继续微调间距，只需要改顶部三个常量
