# 查看应用日志方法

## 当前项目日志标签

当前代码里建议只看这三个唯一标签：

1. `JY_TCP_ARK`
2. `JY_TCP_NATIVE`
3. `JY_ENTRY`

其中最关键的是：

1. `JY_TCP_ARK`：ArkTS 拉起 native TCP 的日志
2. `JY_TCP_NATIVE`：C 语言 TCP 客户端自身日志

## 推荐查看方式

先清空当前日志缓冲区：

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' hilog -r
```

然后重新启动应用，再执行：

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' hilog -t app -T JY_TCP_ARK,JY_TCP_NATIVE,JY_ENTRY
```

这个命令会持续输出实时日志，按 `Ctrl + C` 停止。

## 只看最近 100 行并退出

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' hilog -x -z 100 -t app -T JY_TCP_ARK,JY_TCP_NATIVE,JY_ENTRY
```

## 只看错误日志

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' hilog -x -z 100 -t app -L E,F -T JY_TCP_ARK,JY_TCP_NATIVE,JY_ENTRY
```

## 只看 TCP 相关日志

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' hilog -t app -T JY_TCP_ARK,JY_TCP_NATIVE
```

## 先确认设备是否连接

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' list targets
```

如果能看到设备地址或序列号，说明已经连上。

## 如果要保存日志到设备文件

启动日志落盘：

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' shell hilog -w start
```

日志默认保存在设备目录：

`/data/log/hilog/`

停止落盘：

```powershell
& 'C:\Program Files\Huawei\DevEco Studio\sdk\default\openharmony\toolchains\hdc.exe' shell hilog -w stop
```
